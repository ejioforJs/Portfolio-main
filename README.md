## 💾 Installation

Download the project to your computer and install needed packages with command:

```bash
$ npm install
```

Then create .env file:

```bash
SITE_URL='https://your-domain.com' ## Used for metadata and sitemap generation
CONTACT_EMAIL='ejioforjames12@gmail.com' ## Address that receives contact form messages
RESEND_API_KEY='your_resend_api_key' ## Required to send email through Resend
```

Then start the project on the local server with the command:

```bash
$ npm run dev
```

and open http://localhost:3000 to see this app.
